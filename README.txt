This plugin allows you to easly create your Bootstrap Theme in WordPress.

== Description ==
Theme Creator for Bootstrap allows you to create your own WordPress theme created with Bootstrap,
you can choose among many layouts, colors and more. 

As It's created using Bootstrap, your theme will be completely
Responsive: it will always adapt to the device, offering the best user experience in every situation. 

Using the plugin is very easy, just install and follow the Getting Started instructions.

== Prerequirements ==
Make sure your WordPress folder '/wp-content/themes' is writable, as the Plugin will create a new directory containing all theme files.
